# 2. 📚 核心知识点总结

## 2.1 基础概念
**RCE（远程代码执行）**
- 定义：攻击者可远程执行任意系统命令
- 成因：输入未过滤 / 远程调用漏洞

**WebShell 定义**
- 常见类型：一句话木马、管理型 Webshell
- 检测方式：
  - 异常文件（后缀、创建时间）
  - Web IDS 行为监测
  - 访问日志中的可疑参数
